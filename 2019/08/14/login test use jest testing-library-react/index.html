1. 登录页面场景

  • 进入登录页面时，获取验证码，接口返回一个imageId和content。
  • 用户输入登录名，密码，和验证码。
  • 用户点击登录，调用验证码是否正确接口，返回imageVerification为true时，说明验证码填写正确。
  • 然后请求获取token接口，返回用户信息和其他相关信息，将返回信息存入localstorage.
